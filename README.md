# AquaX Smart Aquarium Monitoring System

Real Time Aquarium monitoring system that monitors the pH level, Turbidity, Temperature and fish behaviour via a camera module.

Hardware Components - Raspberry Pi model 3b+, Sensors, Camera Module, Arduino Analog to Digital converter.

Frontend is done by Chethin and Vidun.

Backend is done by Dewmini.

IoT part is done by Chethin.

AI camera detection part is done by Vidun and Kalaru.

AI chatbot is done by Chethin.

Design database architecture and implement data analyzing part is done by Hasamith and Sahan.

Mobile App Features - Real time sensor data display, Notification Alerts, realtime camera feed.

Tech Stack - react native, java script, python
